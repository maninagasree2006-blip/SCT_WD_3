# Interactive Quiz Application

## 📌 Project Overview

The Interactive Quiz Application is a web-based quiz game developed using HTML, CSS, and JavaScript. Users can answer multiple-choice questions, track their progress, and view their final score at the end of the quiz.

## 🚀 Features

* Multiple-choice quiz questions
* Interactive user interface
* Progress tracking
* Score calculation
* Restart quiz functionality
* Responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

Quiz-Game-Application/

├── index.html

├── style.css

├── script.js


## 🎯 Learning Outcomes

* DOM Manipulation
* Event Handling in JavaScript
* Responsive Web Design
* Interactive UI Development

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Start answering the quiz questions.
